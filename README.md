# From Guidance to Surveillance: An Ethical Framework for Analyzing Data Collection Policies and Practices in Google Maps

## Overview
This repository contains our case study report for CSC304 at King Saud University.

The study examines the ethical implications of Google Maps, focusing on:

- Privacy and personal data collection
- Digital surveillance and public trust
- Informed consent and transparency
- Cognitive dependence on GPS technology

The research applies ethical frameworks and literature-based qualitative analysis to evaluate how location-based technologies affect user autonomy, privacy, and long-term trust.

## Abstract
Google Maps is one of the most widely used navigation tools, relying on GPS, user history, and real-time data to provide efficient routing. While these features improve convenience, they also raise important ethical concerns related to privacy, surveillance, transparency, and cognitive dependence. This study analyzes these challenges and discusses recommendations for responsible data governance and ethical system design.
